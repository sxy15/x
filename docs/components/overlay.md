### Props
| 名称 | 类型 | 默认值 | 说明 | 是否必填 |
| --- | --- | --- | --- | --- |
| show | boolean | false | 是否显示 | 是 |
| zIndex | number | 1 | 层级 | 否 |
| duration | number | 0.3 | 动画时长 | 否 |
| className | string | - | 类名 | 否 |
| lazyRender | boolean | true | 是否懒加载 | 否 |
| customStyle | CSSProperties | - | 自定义样式 | 否 |

| click | - | 点击事件 |
